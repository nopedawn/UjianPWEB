## Backend

**Setup Backend**

```
cd backend/
npm install
npm install mysql express cors nodemon
```

**Run Backend Terminal**
```
npm start
```


## Frontend

**Setup Frontend**

```
cd frontend/
npm install
npm install axios bootstrap react-router-dom
```

**Run Frontend Terminal**

```
npm run dev
```
